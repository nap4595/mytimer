# MultiTimer - 동시 다중 타이머

직관적인 다중 타이머 웹 애플리케이션으로, 최대 15개의 타이머를 동시에 관리할 수 있습니다.

## 📋 주요 기능

### 🔢 다중 타이머 지원
- **5개/10개/15개 타이머** 프리셋 지원
- 각 타이머 독립적 동작 및 제어
- 고정된 CSS Grid 레이아웃 (5개씩 한 줄)

### ⏱️ 직관적인 시간 설정
- **드래그**: 막대를 드래그하여 시간 설정 (10초 단위 스냅)
- **클릭**: 시간 표시를 클릭하여 정밀 입력 (분:초 형식)
- **일괄 적용**: 모든 타이머에 동일한 시간 설정

### 🎨 시각적 피드백
- 20가지 구별되는 타이머 색상
- 진행률 바를 통한 시각적 상태 표시
- 완료 시 깜빡임 효과 (1초 간격)

### 🔊 알림 시스템
- **다양한 종료음**: 벨, 차임, 비프, 딩, 알림 5가지 사운드
- **진동 알림**: 지원 기기에서 진동 피드백
- **Web Audio API** 기반 고품질 사운드

### ⚙️ 사용자 편의 기능
- 개별 타이머 라벨링 지원
- 자동 시작 옵션
- 풀스크린 모드
- 모든 설정 자동 저장 (localStorage)

### 📱 모바일 최적화
- **반응형 레이아웃**: 모바일에서 화면 전체 활용
- **상단 상태바**: 실행 중인 타이머 개수, 음향/진동/전체화면 토글
- **하단 컨트롤바**: 터치 친화적인 전체 시작/정지/리셋 버튼
- **아이콘 중심 UI**: 언어에 관계없이 직관적인 조작
- **슬라이드 설정 패널**: 필요시에만 나타나는 깔끔한 설정 화면

## 🎯 사용법

### 기본 사용법
1. **타이머 개수 선택**: 왼쪽 패널에서 5개/10개/15개 선택
2. **시간 설정**: 
   - 막대 드래그로 빠른 설정
   - 시간 표시 클릭으로 정밀 입력
3. **라벨 설정**: 각 타이머에 "파스타", "운동" 등 이름 부여
4. **실행**: 개별 또는 전체 시작 버튼으로 타이머 실행

### 키보드 단축키
- **스페이스바**: 전체 타이머 시작/정지
- **ESC**: 풀스크린 해제 또는 모달 닫기

### 고급 기능
- **최대 시간 설정**: 10분~60분 범위 선택
- **자동 시작**: 시간 설정 시 바로 실행
- **종료음 선택**: 5가지 알림음 중 선택
- **풀스크린 모드**: 집중 모드를 위한 전체 화면

## 🎨 사용 시나리오

### 요리용 다중 타이머
```
파스타 (8분) + 소스 (12분) + 빵 굽기 (15분)
→ 각각 다른 색상과 알림음으로 구분
```

### 운동용 인터벌 타이머
```
팔굽혀펴기 (2분) + 스쿼트 (2분) + 플랭크 (1분) + 휴식 (1분)
→ 순차적 실행으로 인터벌 트레이닝
```

### 업무용 타임 관리
```
회의 준비 (10분) + 발표 (20분) + Q&A (10분)
→ 시간별 세션 관리로 효율적인 미팅
```

## 🔧 기술 스택

### 프론트엔드
- **HTML5**: 시맨틱 마크업 및 접근성 준수
- **CSS3**: Grid Layout, Flexbox, Custom Properties
- **Vanilla JavaScript**: ES6+ 클래스 기반 모듈화

### 핵심 기술
- **CSS Grid**: 반응형 타이머 레이아웃
- **Web Audio API**: 고품질 알림음 생성
- **Vibration API**: 모바일 진동 피드백

- **AbortController**: 메모리 누수 방지
- **CSS will-change**: 애니메이션 최적화

### 자동 배포
이 프로젝트는 GitHub Pages를 통해 자동 배포됩니다:

1. **GitHub 저장소 생성**: `mytimer` 저장소를 퍼블릭으로 생성
2. **코드 푸시**: 
   ```bash
   git remote add origin https://github.com/[username]/mytimer.git
   git branch -M main
   git push -u origin main
   ```
3. **GitHub Pages 활성화**:
   - Settings → Pages
   - Source: "Deploy from a branch" 
   - Branch: "main" / root
4. **접속**: `https://[username].github.io/mytimer`
